`steep.cloud.ssh.username`

> Username for SSH access to VMs. Can be overriden by the `sshUsername` property
> in each <ScrollLink href="#setups">setup</ScrollLink>. May even be `null` if
> all setups define their own username.

`steep.cloud.ssh.privateKeyLocation`

> Location of a private key to use for SSH
