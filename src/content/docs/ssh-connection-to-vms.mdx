Username for SSH access to VMs. Can be overridden by the `sshUsername` property
in each <DocsLink href="#setups">setup</DocsLink>. May even be `null` if
all setups define their own username.
</dd>

<dt>`steep.cloud.ssh.privateKeyLocation`</dt>
<dd>
Location of a private key to use for SSH
