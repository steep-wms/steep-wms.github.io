This data model describes an additional volume that can be attached to a
virtual machine specified by a <DocsLink href="#setups">setup</DocsLink>.

| Property                               | Type    | Description
| -------------------------------------- | ------- | -----------
| sizeGb<br/>*(required)*                | number  | The volume's size in gigabytes
| type<br/>*(optional)*                  | string  | Type the volume's type. By default, the type will be selected automatically.
| availabilityZone<br/>*(optional)*      | string  | The availability zone in which to create the volume. By default, it will be created in the same availability zone as the VM to which it will be attached.

<CodeExample title="Example:">

```json code-example
{
  "sizeGb": 50,
  "type": "SSD",
  "availabilityZone": "nova"
}
```

</CodeExample>
