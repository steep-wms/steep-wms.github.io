Get details about a single <ScrollLink href="#process-chains">process chain</ScrollLink>
from the database.

Some properties such as `startTime`, `status`, or `endTime` depend on a specific
process chain run (see the section on <ScrollLink href="#process-chains">process chains</ScrollLink>
for more information about runs). This endpoint always displays the latest run
(if available). You can get more information through
the endpoints <ScrollLink href="#get-process-chain-runs">`/processchains/:id/runs`</ScrollLink>
and <ScrollLink href="#get-process-chain-run-by-run-number">`/processchains/:id/runs/:runNumber`</ScrollLink>.

| Resource URL |
| ------------ |

```http
/processchains/:id
```

| Parameters |                                    |
| ---------- | ---------------------------------- |
| id         | The ID of the process chain to return |

| Status codes |                              |
| ------------ | ---------------------------- |
| 200          | The operation was successful |
| 404          | The process chain was not found  |

| Example request |
| --------------- |

```http
GET /processchains/akukkcsv575splk3v2na HTTP/1.1
```
