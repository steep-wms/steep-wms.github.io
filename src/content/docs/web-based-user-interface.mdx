Steep has a web-based user interface that allows you to monitor the execution
of running workflows, process chains, agents, and VMS, as well as to browse
the database contents.

Start Steep and visit any of its <DocsLink href="#http-endpoints">HTTP endpoints</DocsLink>
with your web browser to open the user interface.
