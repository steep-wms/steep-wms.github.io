`steep.db.driver`

> The database driver

> **Valid values:** `inmemory`, `postgresql`, `mongodb`

> **Default:** `inmemory`

`steep.db.url`

> The database URL

`steep.db.username`

> The database username (only used by the `postgresql` driver)

`steep.db.password`

> The database password (only used by the `postgresql` driver)
