Get details about a single <ScrollLink href="#submissions">submission</ScrollLink>
from the database.

| Resource URL |
| ------------ |

```http
/workflows/:id
```

| Parameters |                                    |
| ---------- | ---------------------------------- |
| id         | The ID of the submission to return |

| Status codes |                              |
| ------------ | ---------------------------- |
| 200          | The operation was successful |
| 404          | The submssion was not found  |

| Example request |
| --------------- |

```http
GET /workflows/akpm6yojjigral4cdxgq HTTP/1.1
```
