A default <DocsLink href="#retry-policies">retry policy</DocsLink> that
should be used within a workflow unless a more specific retry policy is
defined elsewhere.

| Property                            | Type     | Description
| ----------------------------------- | -------- | -----------
| processChains<br/>*(optional)*      | object   | An optional default <DocsLink href="#retry-policies">retry policy</DocsLink> that should be applied to every generated process chain

<CodeContainer title="Example">
```json generate-yaml
{
  "processChains": {
    "maxAttempts": 5,
    "delay": "1s",
    "exponentialBackoff": 2,
    "maxDelay": "10s"
  }
}
```
</CodeContainer>
