An agent represents an instance of Steep that can execute <ScrollLink href="#process-chains">process chains</ScrollLink>.

| Property                               | Type    | Description
| -------------------------------------- | ------- | -----------
| id<br/>*(required)*                    | string  | A unique agent identifier
| available<br/>*(required)*             | boolean | `true` if the agent is currently idle and new process chains can be assigned to it, `false` if it is busy executing a process chain
| capabilities<br/>*(required)*          | array   | A set of strings specifying capabilities the agent provides. See also <ScrollLink href="#setups">setups</ScrollLink>.
| startTime<br/>*(required)*             | string  | An ISO 8601 timestamp denoting the date and time when the agent has started.
| stateChangedTime<br/>*(required)*      | string  | An ISO 8601 timestamp denoting the date and time when the value of the `available` property changed from `true` to `false` (i.e. when the agent became busy) or from `false` to `true` (when it became available)

```json code-example
}
```
