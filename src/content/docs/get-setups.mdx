Get information about all configured <DocsLink href="#setups">setups</DocsLink>.
To get a single setup, use
the <DocsLink href="#get-setup-by-id">GET setup by ID endpoint</DocsLink>.

| Resource URL |
| ------------ |

```http
/setups
```

| Parameters              |                              |
| ----------------------- | ---------------------------- |
| None                    |                              |

| Status codes |                                                                |
| ------------ | -------------------------------------------------------------- |
| 200          | The operation was successful                                   |
| 404          | Setups are unavailable because <DocsLink href="#cloud-connection">cloud configuration</DocsLink> is disabled |

| Example request |
| --------------- |

```http
GET /setups HTTP/1.1
```
