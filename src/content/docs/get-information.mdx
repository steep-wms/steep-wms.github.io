Get information about Steep. This includes:

* Steep's version number
* A build ID
* A SHA of the Git commit for which the build was created
* A timestamp of the moment when the build was created

| Resource URL |
| ------------ |

```http
/
```

| Parameters   |                              |
| ------------ | ---------------------------- |
| None         |                              |

| Status codes |                              |
| ------------ | ---------------------------- |
| 200          | The operation was successful |

| Example request |
| --------------- |

```http
GET / HTTP/1.1
```

| Example response |
| ---------------- |

```http
HTTP/1.1 200 OK
content-encoding: gzip
content-length: 130
content-type: application/json

{
    "build": "550",
    "commit": "bc4aef82e3a0f6b2d3a6e9a799eb26a07ba921af",
    "timestamp": 1643191021901,
    "version": "5.11.0"
}
```
