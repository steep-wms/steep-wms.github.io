Get information about Steep. This includes:

* Steep's version number
* A build ID
* A SHA of the Git commit for which the build was created
* A timestamp of the moment when the build was created

| Resource URL |
| ------------ |

```http
/
```

| Parameters   |                              |
| ------------ | ---------------------------- |
| None         |                              |

| Status codes |                              |
| ------------ | ---------------------------- |
| 200          | The operation was successful |

| Example request |
| --------------- |

```http
GET / HTTP/1.1
```

| Example response |
| ---------------- |

```http
HTTP/1.1 200 OK
content-encoding: gzip
content-length: 131
content-type: application/json

{
    "build": "645",
    "commit": "4029e89f642176a6564867529b5150306335e833",
    "timestamp": 1646918799617,
    "version": "5.12.0"
}
```
