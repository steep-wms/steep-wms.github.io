Get information about Steep. This includes:

* Steep's version number
* A build ID
* A SHA of the Git commit for which the build was created
* A timestamp of the moment when the build was created

| Resource URL |
| ------------ |

```http
/
```

| Parameters   |                              |
| ------------ | ---------------------------- |
| None         |                              |

| Status codes |                              |
| ------------ | ---------------------------- |
| 200          | The operation was successful |

| Example request |
| --------------- |

```http
GET / HTTP/1.1
```

| Example response |
| ---------------- |

```http
HTTP/1.1 200 OK
Content-Encoding: gzip
Content-Length: 131
Content-Type: application/json

{
    "version": "6.9.0",
    "build": "2430",
    "commit": "f15073ed7c0b79ddb9441897cab13892bb6548e9",
    "timestamp": 1721019753887
}
```
