Get information about Steep. This includes:

* Steep's version number
* A build ID
* A SHA of the Git commit for which the build was created
* A timestamp of the moment when the build was created

| Resource URL |
| ------------ |

```http
/
```

| Parameters   |                              |
| ------------ | ---------------------------- |
| None         |                              |

| Status codes |                              |
| ------------ | ---------------------------- |
| 200          | The operation was successful |

| Example request |
| --------------- |

```http
GET / HTTP/1.1
```

| Example response |
| ---------------- |

```http
HTTP/1.1 200 OK
content-encoding: gzip
content-length: 130
content-type: application/json

{
    "build": "968",
    "commit": "1cc01e709b4cfd87cf9ee905ed6b75ac490e1738",
    "timestamp": 1656059451391,
    "version": "6.0.0"
}
```
