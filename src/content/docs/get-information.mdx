Get information about Steep. This includes:

* Steep's version number
* A build ID
* A SHA of the Git commit for which the build was created
* A timestamp of the moment when the build was created

| Resource URL |
| ------------ |

```http
/
```

| Parameters   |                              |
| ------------ | ---------------------------- |
| None         |                              |

| Status codes |                              |
| ------------ | ---------------------------- |
| 200          | The operation was successful |

| Example request |
| --------------- |

```http
GET / HTTP/1.1
```

| Example response |
| ---------------- |

```http
HTTP/1.1 200 OK
content-encoding: gzip
content-length: 132
content-type: application/json

{
    "build": "274",
    "commit": "15ba43d39db76f54dfea2badf75cd9c8d1073102",
    "timestamp": 1626944905125,
    "version": "5.8.2"
}
```
