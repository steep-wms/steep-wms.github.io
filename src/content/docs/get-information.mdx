Get information about Steep. This includes:

* Steep's version number
* A build ID
* A SHA of the Git commit for which the build was created
* A timestamp of the moment when the build was created

| Resource URL |
| ------------ |

```http
/
```

| Parameters   |                              |
| ------------ | ---------------------------- |
| None         |                              |

| Status codes |                              |
| ------------ | ---------------------------- |
| 200          | The operation was successful |

| Example request |
| --------------- |

```http
GET / HTTP/1.1
```

| Example response |
| ---------------- |

```http
HTTP/1.1 200 OK
Content-Encoding: gzip
Content-Length: 130
Content-Type: application/json

{
    "version": "6.7.0",
    "build": "2024",
    "commit": "2c83c13ff96d776c298e0c8f69b6f40d7e729ed2",
    "timestamp": 1697633365619
}
```
