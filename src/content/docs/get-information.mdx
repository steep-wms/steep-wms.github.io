Get information about Steep. This includes:

* Steep's version number
* A build ID
* A SHA of the Git commit for which the build was created
* A timestamp of the moment when the build was created

| Resource URL |
| ------------ |

```http
/
```

| Parameters   |                              |
| ------------ | ---------------------------- |
| None         |                              |

| Status codes |                              |
| ------------ | ---------------------------- |
| 200          | The operation was successful |

| Example request |
| --------------- |

```http
GET / HTTP/1.1
```

| Example response |
| ---------------- |

```http
HTTP/1.1 200 OK
content-encoding: gzip
content-length: 132
content-type: application/json

{
    "build": "1020",
    "commit": "0ed0bc04645b82de523eb7b0dd5f77742e340f5d",
    "timestamp": 1658046958016,
    "version": "6.2.0"
}
```
