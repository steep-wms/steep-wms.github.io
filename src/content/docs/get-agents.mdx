Get information about all <DocsLink href="#agents">agents</DocsLink>
currently connected to the cluster. In order to get details about a single agent,
use the <DocsLink href="#get-agent-by-id">GET agent by ID endpoint</DocsLink>.

| Resource URL |
| ------------ |

```http
/agents
```

| Parameters              |                              |
| ----------------------- | ---------------------------- |
| None                    |

| Status codes |                                                                |
| ------------ | -------------------------------------------------------------- |
| 200          | The operation was successful                                   |

| Example request |
| --------------- |

```http
GET /agents HTTP/1.1
```

| ---------------- |

