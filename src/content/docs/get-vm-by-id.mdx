Get details about a single <ScrollLink href="#vms">VM</ScrollLink>
from the database.

| Resource URL |
| ------------ |

```http
/vms/:id
```

| Parameters |                                    |
| ---------- | ---------------------------------- |
| id         | The ID of the VM to return |

| Status codes |                              |
| ------------ | ---------------------------- |
| 200          | The operation was successful |
| 404          | The VM was not found  |

| Example request |
| --------------- |

```http
GET /vms/akvn5rmvrozqzj5k3n7a HTTP/1.1
```
