This data model describes virtual machines created by Steep's cloud manager.

| Property                               | Type    | Description
| -------------------------------------- | ------- | -----------
| id<br/>*(required)*                    | string  | A unique VM identifier
| externalId<br/>*(optional)*            | string  | An identifier generated by the Cloud platform
| ipAddress<br/>*(optional)*             | string  | The VM's IP address
| setup<br/>*(required)*                 | object  | The <DocsLink href="#setups">setup</DocsLink> used to create this VM
| creationTime<br/>*(optional)*          | string  | An ISO 8601 timestamp denoting the date and time when the VM was created. This property is `null` if the VM has not been created yet.
| agentJoinTime<br/>*(optional)*         | string  | An ISO 8601 timestamp denoting the date and time when a Steep agent has been deployed to the VM and has joined the cluster. This property is `null` if the agent has not joined the cluster yet.
| destructionTime<br/>*(optional)*       | string  | An ISO 8601 timestamp denoting the date and time when the VM was destroyed. This property is `null` if the VM has not been destroyed yet.
| status<br/>*(required)*                | string  | The <DocsLink href="#vm-status">status</DocsLink> of the VM
| reason<br/>*(optional)*                | string  | The reason why the VM has the current status (e.g. an error message if it has the `ERROR` status or a simple message indicating why it has been `DESTROYED`)
