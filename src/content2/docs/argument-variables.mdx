An argument variable holds the value of an <DocsLink href="#arguments">argument</DocsLink>.

| Property               | Type   | Description
| ---------------------- | ------ | -----------
| id<br/>*(required)*    | string | The variable's unique identifier
| value<br/>*(required)* | string | The variable's value

<CodeExample title="Example:">

```json code-example
{
  "id": "data_mount",
  "value": "/data:/data"
}
```

</CodeExample>
